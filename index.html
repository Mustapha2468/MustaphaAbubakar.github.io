SQL HOTEL DATABASE

In this project, I undertook a comprehensive analysis of hotel data, encompassing data retrieval, SQL query development, and data visualization using Power BI. The project began with the creation of a database, which involved importing data from multiple sources ('2018$', '2019$', '2020$') into a centralized repository. Subsequently, I crafted SQL queries to extract and transform the data, facilitating insightful analysis. By leveraging SQL's capabilities, I performed data joins, aggregations, and calculations to derive key metrics such as hotel revenue. Following the data preparation phase, I seamlessly connected Power BI to the database, enabling direct access to the structured data.
